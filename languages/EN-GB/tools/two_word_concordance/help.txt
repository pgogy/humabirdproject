<h2>Using the Two word concordancer</h2>
<p>First you need to add or create a <a href="?data=plain_text">plain text file</a></p>
<p>Once you've created a plain text file please then use the concordance tool. You can select the text file you've created, enter two terms to search for.</p>
<p>You also then specify the number of surrounding words to look in for the second term</p>
